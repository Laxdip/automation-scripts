# 🌳 Tree to ZIP

**Paste a folder tree → Get a ZIP file. That's it.**

## Quick Start
```
python3 tree_to_zip.py
```
# Paste your tree, type END, done.

## Example
```
Input:
project/
├── src/
│   └── main.py
├── README.md
└── config.yaml
END

Output: output.zip with all folders + empty files.
```
---

## Why
Stop creating 50 folders manually. Paste and go.
